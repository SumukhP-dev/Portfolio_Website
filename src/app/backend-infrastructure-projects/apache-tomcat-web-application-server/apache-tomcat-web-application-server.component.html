Apache Tomcat Web Application Server

The Apache Tomcat Web Application Server project demonstrates the setup, configuration, and deployment of a Java-based web server. It includes installing the latest Java SDK, configuring Apache Tomcat 11, deploying a custom hello-world style app, and monitoring latency in Chrome DevTools.

This hands-on project highlights skills in Java server administration, deployment pipelines, and debugging web application latency at the network level.
