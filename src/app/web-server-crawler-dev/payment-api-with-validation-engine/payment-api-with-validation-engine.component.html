Payment API with Validation Engine

The Payment Validation Engine project demonstrates the integration of a native C++17 validation library with a C# ASP.NET Core Web API. It includes building a modular validation engine in C++ as a DLL, exposing rules for payment amount, customer name, and currency checks, and calling these native functions from the API using P/Invoke (DllImport).

This hands-on project highlights skills in cross-language integration, Web API development, and designing modular rule-based engines. It also demonstrates knowledge of modern C++ best practices and interop with enterprise-scale .NET applications .
