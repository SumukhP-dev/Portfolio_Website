The Payment API with Validation Engine project showcases a modular C++17 validation engine built as a DLL and seamlessly integrated with a C# ASP.NET Core Payment API via P/Invoke (DllImport). It supports pluggable validation rules including amount, currency, and name validation, and exposes a reusable ValidatePayment function for cross-language interoperability.

This hands-on project highlights skills in native C++ development, .NET interop integration, cloud deployment with AWS Elastic Beanstalk & RDS, and building a payment processing API with robust validation and error reporting.
