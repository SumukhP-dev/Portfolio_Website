Voting Ledger Fabric Application

The Voting Ledger Based System is a blockchain-inspired project built on Hyperledger Fabric. It provides a secure, decentralized way to record and track votes on a distributed ledger, ensuring transparency and immutability.

Key functionality includes vote initialization, submitting new votes, querying individual vote records, and retrieving all stored votes. The system highlights skills in Hyperledger Fabric, Node.js, Go, and ledger-based system design.
