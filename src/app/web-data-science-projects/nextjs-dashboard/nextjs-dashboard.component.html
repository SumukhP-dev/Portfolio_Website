NextJS Finance Dashboard Website

The Next.js Finance Dashboard Website is a full-stack web application designed for invoice management and financial tracking. Users can create accounts, log in securely, and use the dashboard to review revenue summarized in bar charts. Features include new invoice creation, customer records, and a real-time invoice overview.

Built with Next.js + TailwindCSS, using Next.js Server Actions and a Neon PostgreSQL database, the project highlights skills in modern frontend frameworks, serverless backends, and data visualization.
