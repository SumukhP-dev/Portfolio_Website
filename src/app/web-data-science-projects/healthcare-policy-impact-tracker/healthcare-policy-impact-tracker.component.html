Healthcare Policy Impact Tracker

The Healthcare Policy Impact Tracker is a data science and full-stack project designed to analyze how California healthcare policies affect public health outcomes. Leveraging Python (Pandas) for dataset preparation, ARIMA models for historical trend analysis, and Gaussian Processes for forecasting, the project provides decade-long projections of mortality and healthcare access trends. Deliverables include an interactive web app for policymakers and the public and an automated analysis pipeline. This project demonstrates skills in data science, forecasting, full-stack development, and visualization.
